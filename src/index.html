Assumptions

Everything Excel is missing for big workbooks

Sheet Navigator covers what tools like Kutools do for sheet management — without the toolbar bloat and without a paid license.

⌕ Accent-insensitive search

Type "resume" to find "Résumé". Type "key 24" to find "Key 2024". Filter your sheet list instantly, even with 200+ entries.

★ Per-workbook pinning

Pin the sheets you visit constantly. Pins are stored inside the workbook itself, so they survive close/open cycles and follow the file when shared with a colleague.

▦ Folder hierarchy via /

Name a sheet Region/France and it appears as a child of an expandable "Region" folder. Drag between folders to rename automatically. Persisted collapse state per workbook.

⇅ Drag-and-drop reorder

Reorder sheets from the side panel. Much faster than the bottom tabs, especially when there are thirty sheets and you have to scroll horizontally.

⊘ Hidden sheets, your call

Show or hide the hidden sheets in one click. Toggle visibility per-sheet via the eye icon. Useful for workbooks with lots of intermediate calculation sheets.

⟲ Real-time sync

The list updates automatically as you add, delete, rename or activate sheets. No manual refresh, no stale state.

Cross-platform, no heavy install

Sheet Navigator is a modern Office Add-in. It runs in a web sandbox embedded in Excel — no system installer, no admin privileges.

✓Excel Mac ✓Excel Windows ✓Excel for the Web ✓Excel iPad

No VBA script, no .xlam, no macro to unblock. The panel appears in the Home ribbon as soon as the manifest is installed.
